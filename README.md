### UniLift 2016  


__Для установки QT 5 используйте__:  
```bash
pip3 install pyqt5
```


__Сочетания клавиш__:
* q - выход
* 1, 2, ..., 9 - нажатие кнопки вызова лифта на соответствующем этаже
* Ctrl + 1, 2, ..., 9 - нажатие кнопки соответствующего этажа внутри лифта
